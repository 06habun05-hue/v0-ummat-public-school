# UI Guidelines

This document locks in the exact spacing, sizing, and styling tokens needed to maintain our premium "wow" factor. Do not deviate from these standards.

## Typography Hierarchy
* **Page Titles (H1):** `text-3xl sm:text-4xl font-heading font-black tracking-tight`
* **Section Titles (H2):** `text-xl sm:text-2xl font-heading font-bold`
* **Body Text (P):** `text-sm sm:text-base font-sans leading-relaxed`
* **Micro-labels/Badges:** `text-[10px] font-black uppercase tracking-widest` (This is the strict standard for giving the UI a premium enterprise look).

## Spacing, Padding & Layout
* **Cards/Containers:** Must use `p-6` or `p-8` with `rounded-3xl` and `border-border` to maintain a spacious, modern feel.
* **Primary Buttons:** Must be chunky and prominent. Use `h-12 px-8 rounded-2xl` with `shadow-xl shadow-primary/20`.
* **Standard Inputs:** `h-12 rounded-2xl bg-background border-border shadow-inner`.
* **Icons:** Use `lucide-react`. Size `16` for inline text/buttons. Size `20` or `24` for standalone icons in cards.
