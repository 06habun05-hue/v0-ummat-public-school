# Lingo & Vocabulary Guidelines

## Core Rule
Keep the reading level at or below the 8th grade. The UI must be instantly understandable by all staff, teachers, and admins regardless of their technical background. **Clarity always beats complexity.**

## Banned Corporate Jargon & Approved Alternatives

| Banned Jargon | Approved Alternative |
| --- | --- |
| Financial Treasury | Fee Management |
| Orchestrating institutional liquidity | Track and manage payments |
| System Provisioning | Create Account / Add User |
| Cumulative Revenue | Total Collected |
| Outstanding Liquidity | Pending Fees |
| Collection Velocity | Collection Rate |
| Pedagogical Assessment | Assessment |
| Remittance / Monetary Compensation | Payment / Fees |
| Authenticate | Log In / Sign In |
| System Exception | Error / Crash |

## Actionable UI Text
Buttons and links must use clear, one-to-two word action verbs.
* **Good:** Save, Pay, Add User, Delete, Export Report
* **Bad:** Execute Transaction, Finalize Data Entry, Provision Entity
